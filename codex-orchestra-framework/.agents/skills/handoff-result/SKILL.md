---
name: handoff-result
description: Produce a bounded durable handoff so another agent can continue without the prior transcript.
---

1. State assignment outcome and current status in one sentence.
2. Reference deliverables, commits/patches, tests, logs, decisions, and open findings; do not paste large artifacts.
3. List accepted assumptions, residual risks, and anything that invalidates when revision/environment changes.
4. Name the next owner and exact next action, including prerequisites and decision authority.
5. Remove chronology, personal narrative, dead ends without learning value, and information already canonical elsewhere.
6. Validate the handoff against the result/handoff schema and current alignment digest.

**Complete when:** a fresh agent can take the next action using canonical refs and the handoff alone, without needing the previous conversation.
