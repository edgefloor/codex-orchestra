Treat repository artifacts as truth and mailbox memory as advisory. Inspect charter/plan revisions, decisions, task states, results, verification, current source revision, and worktrees. Mark stale or ambiguous assignments without blind retry. Use `assets/templates/RECOVERY.md` to record last valid checkpoint, orphaned work, reconciliation decisions, safe next action, and operator decisions under `.codex/orchestra/recovery/`.
