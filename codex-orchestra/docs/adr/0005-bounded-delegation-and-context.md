# Bound delegation and pass Context Capsules instead of transcripts

Every delegated branch must have a purpose, authority, child and attempt budget, Join Owner, output contract, stop condition, and reserved review capacity. The closest competent parent owns the wait, evidence review, join, and bounded upward synthesis; child completion is only a transport signal and never proves assignment success.

Fresh context is the default. Spawns explicitly request no transcript inheritance when the runtime supports that choice, and every assignment receives a minimal revision-pinned Context Capsule. Raw transcripts and logs stay local unless a cited excerpt is necessary evidence.

## Consequences

- Leaf Workers do not spawn additional agents.
- A Context Capsule contains goal, done condition, deliverables, non-goals, binding references, write domain, source revision, validation, authority, escalation, budget, and result location.
- Standalone legacy Delegation Permit, Role Card, Branch Report, and Portfolio schemas are not mandatory; their essential fields live in task, plan, result, and policy artifacts.
